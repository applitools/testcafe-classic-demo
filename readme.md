## Testcafe Classic Demo
### Usage
1. Run ```npm i ```

2. Set ```APPLITOOLS_API_KEY``` env variable
3. Run tests :
    
    Run locally

    ```npm run test:chrome```

    Run locally with Safari (Mac)

    ```npm run test:safari:sl```

    Run on SauceLabs 

    ```npm run test:safari:sl```
    
    <br/>

* For running individual tests :

    ```npx testcafe chrome:headless tests/ipad-air*```
